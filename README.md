# orders-management-system

## General info
* Java JDK 17
* Desktop GUI Application
* IDE - IntelliJ

## Features
* Clients:
  * Show all Clients
  * Show Client
  * Add new Client
  * Remove Client
* Items:
  * Show all Items
  * Show Item
  * Add new Item
  * Remove Item
  * Filter by Item name
* Orders:
  * Show all Orders
  * Show Order
  * Add new Order
  * Remove Order
  * Filter by:
    * Ordered At range
    * Gross Total range
    * Client

## Technical features
* Data storage - TXT files
